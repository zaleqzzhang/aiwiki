## 简介

LangChain 创始人，AI Agent 架构领域的活跃思考者和实践者。在 X (@hwchase17) 上持续分享关于 Agent 架构、持续学习、记忆系统的见解。

## 核心观点

- Agent 持续学习应区分三个层次：模型层、Harness 层、上下文层
- Trace 是所有持续学习流程的核心数据源
- Agent 生态的三层分类学：Framework（抽象层）→ Runtime（生产基础设施）→ Harness（开箱即用完整系统）
- 上下文层学习是最灵活、最常用的改进方式
